# 機能
- Sub-6 / ミリ波 / アンカーバンド の検出
- 5Gの場合は スタンドアローン方式 / ノンスタンドアローン方式 の表示
- バックグラウンドでも通知領域から確認
- おまけ程度のバンド表示
    - `n258`とか

(なんか従量制ネットワークか無制限ネットワークか検出出来るよ！って書いてあったんだけど検出できてない。流石にできないやろ...)
